# SocketCluster-Nest

## Description

A [Nest](https://github.com/nestjs/nest) project with [Socketcluster](https://github.com/SocketCluster/socketcluster) as Message bus

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
